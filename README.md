# liminal-website

website for [liminal spaces](http://liminalspaces) built using [DocPad](http://docpad.org).

## how to

### install

```
git clone https://github.com/derekrazo/liminal-website
npm install
```

### run

`npm start`

### develop

`npm run develop`

### deploy

`npm run deploy`
